## 💻 담당 업무 구현 화면
### 예매 ###
![image](https://user-images.githubusercontent.com/83936196/131256207-1fc7097b-9666-434b-bef7-7ad9498aec7e.png)

**📷 예매 1단계 - 영화/ 지역/ 극장/ 날짜/ 상영시간 선택** 
- 영화 포스터를 클릭하여 영화를 선택할 수 있다.
- 지역을 선택하면 해당 지역에 있는 극장들이 출력된다.
- 영화, 지역, 극장명까지 선택하면 상영시간이 조회된다.

![image](https://user-images.githubusercontent.com/83936196/131256210-4419ad5c-4475-4dbb-af51-4c4ab5ffc1b7.png)

**📷 예매 2단계 - 인원/ 좌석 선택** 
- 성인, 청소년, 어린이별로 인원을 선택할 수 있다.
- 선택된 인원만큼 좌석을 선택할 수 있다.
- 인원만큼 자동으로 금액이 계산되어 나타난다.

**📷 매점 상품 구매 내역** 
- 매점에서 구매한 상품 내역을 확인할 수 있다.

<img src="https://user-images.githubusercontent.com/83936196/131256520-809006be-55fc-4ccd-a97f-1c5e052b2e7d.png" width="48%"> <img src="https://user-images.githubusercontent.com/83936196/131256523-aec01595-060e-4d3b-b7a5-5fc5b4aadd6b.png" width="48%">

**📷 회원정보수정** 
- 이름, 아이디, 생년월일을 제외한 항목들을 수정할 수 있다.
- 수정하기 버튼 클릭 시, 비밀번호를 다시 한 번 입력해야 변경이 완료된다.

## 📝 프로젝트 후기
<p>그 동안 배웠던 모든 기능들을 종합하여 프로젝트를 진행하였다. JAVA, Oracle에 HTML, CSS, JavaScript를 활용하여 홈페이지의 디자인을 구성하고, JSP, Servlet, JDBC를 이용하여 세부적인 기능들을 추가했다. 앞서 진행했던 두 프로젝트와는 확연히 다른 볼륨감에 개발하는 내내 시간이 모자르다는 느낌을 받았다. 더불어 내가 잘하는 부분과 약한 부분을 알 수 있던 프로젝트 였다.</p>
<p>지난 몇 달간 수업을 들으며 다양한 언어들을 접하고 배울때마다 이것들이 어떻게 연결이 되는지, 어떻게 사용이 되는지 궁금한 부분들이 종종 있었는데 이번 기회에 몸소 느끼고 이해할 수 있었다. 또 개인적으로 가장 실무와 닮아있는 프로젝트라는 생각이 들었다. 또 나 혼자 하는 작업이 아니기에 팀으로 하는 작업을 서로를 더 배려하고 많이 소통을 해야 좋은 결과물이 나온다고 느꼈다.</p>
